Frequently Asked Questions

Questions & Answers

Everything you need to know about Nexis Infinity's architecture, security framework, and device compatibility.

How does Nexis AI run completely offline? +

Nexis Infinity uses highly optimized, quantized small language models (like Llama-3-Small or Phi-3) that fit inside your phone's storage and execute directly using the mobile device's Neural Processing Unit (NPU).

Is my personal data safe with Nexis? +

Yes, 100%. Nexis is designed with a Privacy-First core. All text processing, voice recognition, and device automation tasks occur locally inside an isolated vault on your device, ensuring zero data transmission to cloud servers.

What is the Zero-Root Accessibility API? +

Unlike old-school root automation scripts, Nexis interacts with your phone via standard, official Android Accessibility permissions. This allows the AI agent to perform safe navigation, system toggles, and clicks without compromising device security.

Does running local AI drain the phone's battery? +

Nexis is highly optimized for modern smartphone chipsets. By leveraging dedicated low-power NPUs instead of heavy CPU processing, battery consumption is minimized to a level similar to running a standard offline mobile game.
