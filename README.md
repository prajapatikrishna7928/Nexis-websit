# 🚀 Nexis Infinity — Next-Gen Autonomous AI & Voice Agent

Nexis Infinity is a powerful, offline-first autonomous AI agent designed for Android devices. Powered by quantized Small Language Models (SLMs) and on-device NPU execution, it ensures 100% privacy with zero cloud telemetry.

## ✨ Key Features
- **⚡ On-Device Execution:** Works completely offline without server latency.
- **🔒 Privacy-First Architecture:** Zero data processing on cloud servers.
- **🎙️ Autonomous Voice Control:** Smart device navigation using official Android Accessibility APIs.
- **⚡ Low Battery Consumption:** Optimized for modern smartphone NPUs.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Modern Glassmorphism UI), Vanilla JavaScript
- **Backend & Database:** Google Firebase (Firestore) for live reviews & feedback
- **Hosting:** Vercel / GitHub Pages
